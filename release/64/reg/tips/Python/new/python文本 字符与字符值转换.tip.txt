python文本 字符与字符值转换
场景：

将字符转换成 ascii 或者 unicode 编码

在转换过程中，注意使用 ord 和 chr 方法

  >>> print(ord('a'))  
  97  
  >>> print(chr(97))  
  a  
  >>>   
有时候需要反转过来使用：

  >>> print(str(ord('a')))  
  97  
  >>> print(chr(ord('a')))  
  a  
  >>>   
这个时候需要注意 str() 与 chr() 之间的区别：str 是把返回的对象直接转为字符串，而 chr 则是把返回的对象转换为对应的 ascii 码的字符串。
