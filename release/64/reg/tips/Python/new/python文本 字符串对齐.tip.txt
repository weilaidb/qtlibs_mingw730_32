python 字符串对齐
场景：

字符串对齐

python 提供非常容易的方法，使得字符串对齐

  >>> print("abc".center (30,'-'))  
  -------------abc--------------                         
  >>> print("abc".ljust (30)+'|')  
  abc                           |  
  >>> print("abc".rjust (30))  
                             abc  
  >>>   
分别是 center，ljust，rjust

三个方法默认填充是空格，也可以使用其他字符填充
