背景
用pip安装依赖包时默认访问https://pypi.Python.org/simple/，但是经常出现不稳定以及访问速度非常慢的情况，国内厂商提供的pipy镜像目前可用的有：

清华：https://pypi.tuna.tsinghua.edu.cn/simple
阿里云：http://mirrors.aliyun.com/pypi/simple/
中国科技大学 https://pypi.mirrors.ustc.edu.cn/simple/
华中理工大学：http://pypi.hustunique.com/
山东理工大学：http://pypi.sdutlinux.org/
豆瓣：http://pypi.douban.com/simple/
note：新版ubuntu要求使用https源，要注意。

pip install package_name;（直接安装；推荐使用，但因网络问题，有时安装失败)
pip install local.whl ; (下载whl文件，然后选择本地安装)
1. 指定镜像源
有两种方式使用我们自己指定的镜像源。

1.1 手动指定
pip install -i http://pypi.douban.com/simple Flask  -- trusted-host pypi.douban.com
或者
pip install Flask -i http://pypi.douban.com/simple -- trusted-host pypi.douban.com
1
2
3
不过这种方式在每次安装时都要手动指定，因此你可以把它写在配置文件中，这就是第二种方法。

1.2 在当前用户目录下创建.pip文件夹。
Linux下，修改 ~/.pip/pip.conf (没有就创建一个文件夹及文件。文件夹要加“.”，表示是隐藏文件夹)

mkdir ~/.pip
1
然后在该目录下创建pip.conf文件填写：

[global]
index-url = https://pypi.tuna.tsinghua.edu.cn/simple

[install]
trusted-host=mirrors.aliyun.com
1
2
3
4
5
上面配置是针对OSX／Linux系统，如果是Windows，那么创建%HOMEPATH%\pip\pip.ini文件来配置。

例如：C:\Users\fxl\pip\pip.ini

pip.ini配置的主要一些配置：

[global]
index-url = https://pypi.tuna.tsinghua.edu.cn/simple

[install]
trusted-host=mirrors.aliyun.com
1
2
3
4
5
新增：一般本地会有多个python版本，创建多个pip.ini文件（与自己的安装时pip命名对应）



2. 安装指定版本的包
要用 pip 安装指定版本的 Python 包，只需通过 == 操作符 指定。

pip install robotframework==2.8.7
1
将安装robotframework 2.8.7 版本。
