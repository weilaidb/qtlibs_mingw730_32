1．Unix
对于Unix用户，如果你想跳过README里的编译步骤，也可以使用下面的步骤成功编译。我选择CMake来编译Google Mock。在Google Mock安装根目录下（即$GMOCK_HOME），执行下列步骤：
mkdir mybuild
cd mybuild
cmake ..

make 
make install

#安装，需要手工拷贝库和头文件
find | grep libgmock.a
cp libgmock.a /usr/local/lib
cp libgtest.a /usr/local/lib

cd ..
find | grep gmock.h
find include/
cp -af include/gmock/ /usr/local/include/
cp -af gtest/include/gtest/ /usr/local/include/


你也需要编译Google Test，它包含在Google Mock里。
cd $GMOCK_HOME/gtest
mkdir mybuild
cd mybuild
cmake ..
make
