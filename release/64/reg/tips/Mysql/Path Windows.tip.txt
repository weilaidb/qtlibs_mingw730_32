#修改密码
②进入mysql，登录
　　　　mysql -u root -p
　　　　不用输入密码，直接回车（出现Enter Password 也一样直接回车，即可登陆成功）

③输入use mysql，修改root的密码：
　　　　update user set authentication_string=password(‘新密码’) where user=‘root’;
　　　　flush privileges;

use mysql
update user set authentication_string=password('123456') where user='root';
flush privileges;
