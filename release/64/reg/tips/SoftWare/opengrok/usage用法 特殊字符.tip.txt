2.1 特殊字符
opengrok在查询语法中支持转义字符；这些特殊字符如“+ - && || ! ( ) { } [ ] ^ " ~ * ? : \ /”。转义这些字符需要使用反斜杠\。例如搜索(1+1):2，完整的搜索字符串如下：

1\+1\+\:2

需要特别注意的是索引字符主要是数字，字母和下划线，一个字符是无法进行索引。
