在上一篇解决冲突的时候，我们观察到当我们点击编辑冲突的时候，会出现三个文件，文件结尾分别时remote，local,base,、

这三个参数是什么意思呢？



local:指的本地要提交，要合并的状态。

base:指的是local和remote共同的节点。

remote:指的是我们从服务器将上一次提交拉到本地对应的状态。
