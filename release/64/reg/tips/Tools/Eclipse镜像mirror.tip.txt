前言
版本： eclipse 2018.09 photon版
步骤：
更换镜像源
window -> preferences -> install/update -> available software sites
把所有的地址的前缀http://download.eclipse.org/ 改成国内镜像地址（如：http://mirror.bit.edu.cn/eclipse/）
把需要用的链接勾选上，我这里是全勾选。
这是一些国内的镜像源地址，注意如果某一个镜像源用不了，换一个镜像源说不定就可以了，因为中科大的镜像我用不了，换成北理工就可以了：
中国科学技术大学 http://mirrors.ustc.edu.cn/eclipse/
北京理工大学  http://mirror.bit.edu.cn/eclipse/
大连东软信息学院  http://mirrors.neusoft.edu.cn/eclipse/

2
3
安装插件，以web插件为例
help -> install new software
work with -> 选择http://mirror.bit.edu.cn/eclipse/releases/2018-09
选择web,xml,java ee and…
next -> next …
最后：希望对大家有用
