csv 是逗号分隔值的缩写，是一种用来存储表格数据的文本文件格式。在 csv 文件中，数据被存储为若干记录，每条记录由若干字段组成，字段之间使用逗号分隔。

xml 是可扩展标记语言的缩写，是一种用来存储结构化数据的文本文件格式。在 xml 文件中，数据被存储为若干元素，每个元素都有一个标签和一些属性，用来描述该元素的含义和内容。

json 是 JavaScript 对象表示法的缩写，是一种用来存储结构化数据的文本文件格式。在 json 文件中，数据被存储为键值对的形式，每个键值对由一个名称(键)和一个值组成。
