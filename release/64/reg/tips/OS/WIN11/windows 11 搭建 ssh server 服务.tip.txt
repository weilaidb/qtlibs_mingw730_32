执行命令
在 OpenSSH 目录下以管理员的身份 执行 powershell.exe -ExecutionPolicy Bypass -File install-sshd.ps1



执行命令 netsh advfirewall firewall add rule name=sshd dir=in action=allow protocol=TCP localport=22



开启 sshd 服务 —> net start sshd



设置服务自启动 —> Set-Service sshd -StartupType Automatic

常见问题
SSH服务器拒绝了密码



分析原因：ssh配置文件的配置不允许root用户进行远程登录



用户名 输入 root 试试
