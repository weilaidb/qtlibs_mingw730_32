properties中输入中文乱码
原因是因为在properties中输入的中文是以UTF-8存储显示的，而在其运行时会将输入的中午转化为ASCII进行使用，所以会出现乱码；所以需要我们进行配置。如有需要也可将中文转化为Unicode去运行程序。

settings -- File Encodings -- Properties Files 
1.UTF-8
2.Transparent navitve-to-ascii conversion
