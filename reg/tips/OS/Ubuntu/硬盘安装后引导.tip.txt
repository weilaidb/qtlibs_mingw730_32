第三步：双系统启动引导
关机后，再次插入Ubuntu启动盘，选择Try Ubuntu without installing，进入试用桌面，点击右上角，进行联网（一定要联网）。

利用boot-repair轻松制作（修复）启动引导。
CTRL +ALT +T 打开终端，执行以下命令以安装boot-repair：

sudo add-apt-repository ppa:yannubuntu/boot-repair
sudo apt-get update
sudo apt-get install boot-repair

2
3
安装完成后，在应用中搜索boot-repair并运行，选择recommand repair，之后程序会自动执行。执行完成后，关机，拔掉U盘，可以看见已经有了开机引导！
